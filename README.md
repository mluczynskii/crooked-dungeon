# crooked-dungeon
Uni class project, simple 2D tboi/zelda inspired dungeon crawler game in Java with sprites and tiles made from scratch.   

Compiling command:
javac -cp . -d .\bin .\main\Window.java

Run command:
java -cp .\bin main.Window
